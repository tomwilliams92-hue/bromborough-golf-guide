# Bromborough GC — Back-Tee Yardage Book

A single-page, mobile-friendly course strategy guide for **Bromborough Golf Club** (Wirral), played off the back tees (White, par 72, 6,548 yds).

Hole-by-hole game plan — tee shot, the miss to avoid, the approach, and club picks — mapped to a specific set of carry yardages, built for a round near level par. Tap any hole number to mark it played (saved locally).

**Live:** https://tomwilliams92-hue.github.io/bromborough-golf-guide/

Course detail sourced from the club's official scorecard and hole-by-hole flyover. Single static `index.html`, no build step.
